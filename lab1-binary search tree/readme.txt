Binary Search Tree program

1. Files includes
- adslab1.cpp: C++ source code 
- bst.txt: Text file for automatic persistent storage

2. How to run (Windows/VS Code)
- Open the terminal in the folder containing 'adslab1.cpp'.
- Compile the code:
    g++ adslab1.cpp -o bst
- Run the executable:
    ./bst

3. Using the menu
    1. Insert: Enter a number to add it to the tree. 
    2. Delete: Enter a number to remove it. 
    3. Search: Enter a number to check if it exists in the tree.
    4. Min/Max: See the smallest and largest numbers in the bst.
    5. View Tree: Prints all numbers in the tree from smallest to largest (In-order).
    6. Exit: Closes the program. Use this before recompiling to avoid file errors.

4. Note: BST logic follows lecture slides; AI was used for terminal UI and file saving features.
